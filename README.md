## üìù Study Techniques (From Class 1)

Using evidence-based learning strategies from "Learning How to Learn":

- ‚úÖ **Active Recall** - Test myself regularly, code without looking at notes
- ‚úÖ **Spaced Repetition** - Review concepts daily using Anki flashcards
- ‚úÖ **Chunking** - Break complex topics into smaller pieces
- ‚úÖ **Pomodoro Technique** - 25 min focused work, 5 min breaks
- ‚úÖ **Interleaved Practice** - Mix different topics instead of mastering one at a time
